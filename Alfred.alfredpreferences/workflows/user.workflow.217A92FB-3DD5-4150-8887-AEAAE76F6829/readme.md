> 程序代码注释


## Install

```
$ npm install --global alfred-comment
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `comment`, <kbd>Enter</kbd>, and your query.
